# Public Assets

## Files needed in this folder:

### 1. profile.jpg
- **Purpose:** Your profile photo that will appear in the hero section
- **Format:** JPG, PNG, or WebP
- **Recommended size:** 400x400 pixels or larger (square ratio)
- **Tips:** Use a professional photo with good lighting

### 2. cv.pdf
- **Purpose:** Your CV/Resume that visitors can download
- **Format:** PDF
- **File name:** Must be exactly `cv.pdf`
- **Tips:** Make sure the file is up-to-date and professional

## How to add your files:

1. Copy your profile photo to this folder
2. Rename it to `profile.jpg` (or `profile.png`)
3. Copy your CV to this folder
4. Rename it to `cv.pdf`

**Note:** If you use PNG format for profile photo, you need to update the src in `components/HeroSection.tsx` from `/profile.jpg` to `/profile.png`
